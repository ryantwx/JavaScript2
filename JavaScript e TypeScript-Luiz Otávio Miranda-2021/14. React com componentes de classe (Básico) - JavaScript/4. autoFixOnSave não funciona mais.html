Depois de uma atualização, o autoFixOnSave parou de funcionar para salvar automaticamente com o eslint. Agora as configurações são:

settings.json do VS Code

{
    // ...
    "editor.codeActionsOnSave": {
        "source.fixAll.eslint": true,
        "source.fixAll": true
    }
    // ...
}

Caso queira ver como é o meu settings.json pessoal, acesse este gist.
